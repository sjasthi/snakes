Puzzle Types

🐍
Snake Puzzle Word Grid

A quote snakes through a letter grid. Trace the hidden path from cell to cell to reveal the message.

  • Easy 10×10 · Normal 15×15 · Hard 20×20
  • Random filler letters camouflage the path
  • Toggleable solution before printing
📊
Drop Quote Column Drop

Letters fall into columns above a blank answer grid. Decide which letter fills each slot to spell the quote.

  • Column widths: 10, 15, 20, or 25
  • Shuffled columns keep every puzzle unique
  • Show/hide solution toggle before printing
🖼️
Rebus Puzzle Image Clues

Each letter of the target word is clued by an image. Identify the picture, find the marked letter, solve the word.

  • Pixabay photos or AI-generated images
  • Full Telugu Rebus via word bank
  • Generate multiple puzzles per word

How It Works

1
Load Quotes

Upload a .txt file or add quotes one by one. Supports English and Telugu.

2
Pick a Puzzle

Navigate to Snake, Drop Quote, or Rebus. Puzzles generate automatically from your library.

3
Set Preferences

Choose difficulty, column width, or image source. Toggle the solution on or off.

4
Print Your Book

Click Print PDF. Every puzzle page breaks cleanly — ready to hand out as a full book.

About

Snakes is a capstone project built for ICS 499 — Software Engineering & Capstone Project, Spring 2026, under Professor Siva Jasthi.

The app takes a collection of quotes and generates a fully printable puzzle book with Snake Word puzzles, Drop Quote puzzles, and Rebus image challenges — with multi-script support for both English and Telugu.

Built by Nathan Vo & Rocky Vang  ·  Python · Flask · Bootstrap 5 · jQuery

  • 🗂️
    Full quote management
    Upload, add, remove, and replace quotes with live CRUD — no page reload required.
  • 🌐
    Telugu & English support
    The Ananya API handles multi-byte Telugu grapheme clusters so puzzles render correctly.
  • 🖨️
    Print-ready output
    Every puzzle page is styled for clean browser printing with automatic page breaks.
